## Dokumentation

### Herunterladen der Software
Nachdem wir die SSH Verbindung zum RPI aufgebaut haben, haben wir den Befehl:
```
sudo apt update
```
verwendet und dann die Software herunter geladen mit folgendem Befehl:
```
sudo apt install sense-hat python3-sense-hat
```
dann haben wir python gestartet und diese Befehle abgesetzt:
```
from sense_hat import SenseHat 
sense = SenseHat()
```
### Python Scripts
Dann haben wir ein Pyhton Programm geschrieben welches verschiedenste Dinge auf den LEDs ausgibt.

```
from sense_hat import SenseHat

from time import sleep

sense = SenseHat()

sense.clear()

print("=== Wird ausgeführt... ===")



# Wird ausgeführt...


print("Teste LED Matrix...")  

colors = [(255,0,0), (0,255,0), (0,0,255), (255,255,0), (255,255,255), (0,255,255)]

for c in colors:

    sense.clear(c)

    sleep(1)

sense.clear()

sense.show_message("Das", text_colour=[0,255,0])

sense.show_message("ist", text_colour=[255,0,0])

sense.show_message("ein", text_colour=[0,255,255])

sense.show_message("SenseHat", text_colour=[0,0,255])

sleep(1)

# Pixel-Test

sense.set_pixel(0, 0, 255, 0, 0)

sense.set_pixel(7, 7, 0, 255, 0)

sleep(1)

sense.clear()
```

from sense_hat import SenseHat

from time import sleep
 
sense = SenseHat()

sense.clear()
 
print("== Wird ausgeführt... ==")
 
# DISPLAY HELLIGKEIT 

sense.low_light = True

sense.show_message("LOW")

sleep(1)

sense.low_light = False

# BILDMUSTER

pixels = []

for i in range(64):

    pixels.append([i*4 % 255, 255 - (i*4 % 255), (i*2) % 255])

sense.set_pixels(pixels)

sleep(2)

sense.clear()

 
print("=== TEST ABGESCHLOSSEN ===")

sense.show_message("DONE", text_colour=[0,255,0])
