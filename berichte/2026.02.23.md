## Übung (msf Basics)

### search -h
```bash
search -h
```
Zeigt die Hilfe zur search-Funktion an.  

### search ssh
```bash
search ssh
```
Durchsucht alle verfügbaren Module nach dem Begriff ssh.


### search scanner/ssh
```bash
search scanner/ssh
```
Sucht gezielt nach Modulen im Verzeichnis scanner/ssh.


### search type:auxiliary name:ssh
```bash
search type:auxiliary name:ssh
```
Filtert die Suche nach Modultyp auxiliary und dem Begriff ssh im Namen.


### info auxiliary/scanner/ssh/ssh_version
```bash
info auxiliary/scanner/ssh/ssh_version
```
Zeigt detaillierte Informationen über dieses Modul an, z. B. Beschreibung, Optionen und Referenzen.


### use auxiliary/scanner/ssh/ssh_version
```bash
use auxiliary/scanner/ssh/ssh_version
```
Lädt das Modul, damit es konfiguriert und ausgeführt werden kann.


### options
```bash
options
```
Zeigt alle einstellbaren Parameter des aktuell geladenen Moduls an.

### set RHOSTS <target IP>
```bash
set RHOSTS 192.168.1.10
```
Setzt die Zieladresse.  
RHOSTS kann eine einzelne IP, mehrere IPs oder ein ganzes Netzwerk sein.


### run
```bash
run
```
Führt das aktuell geladene Modul mit den gesetzten Optionen aus.

Lädt das HTTP-Title-Scanner-Modul. Dieses liest den Titel einer Webseite aus.


### info
```bash
info
```
Zeigt Informationen zum aktuell geladenen Modul an.

### options
```bash
options
```
Zeigt die konfigurierbaren Optionen des HTTP-Moduls an.


### set RHOSTS <target IP>
```bash
set RHOSTS 192.168.1.10
```
Setzt die Zieladresse für den HTTP-Scan.


### run
```bash
run
```
Startet den HTTP-Scan.

### set RHOSTS www.orf.at
```bash
set RHOSTS www.orf.at
run
```
Setzt die Ziel-Domain auf www.orf.at und führt den Scan aus.

### set RHOSTS www.htl-braunau.at
```bash

run
```
Setzt die Ziel-Domain auf www.htl-braunau.at und führt den Scan aus.

## Allgemeine Metasploit Befehle

### msfconsole
```bash
msfconsole
```
Startet die Metasploit-Konsole.

### exit
```bash
exit
```
Beendet Metasploit.

### show exploits
```bash
show exploits
```
Zeigt alle verfügbaren Exploit-Module an.

### show auxiliary
```bash
show auxiliary
```
Zeigt alle Auxiliary-Module an.


### show payloads
```bash
show payloads
```
Zeigt alle verfügbaren Payloads an.

### show options
```bash
show options
```
Zeigt die Optionen des aktuell geladenen Moduls an.

### set <option> <wert>
```bash
set RPORT 22
```
Setzt eine Modul-Option lokal für das aktuelle Modul.

### setg <option> <wert>
```bash
setg LHOST 192.168.1.100
```
Setzt eine Option global für alle Module.

### getg
```bash
getg
```
Zeigt alle global gesetzten Variablen an.

### unset <option>
```bash
unset RHOSTS
```
Entfernt eine gesetzte Option im aktuellen Modul.

### unsetg <option>
```bash
unsetg LHOST
```
Entfernt eine global gesetzte Option.

### exploit
```bash
exploit
```
Startet ein Exploit-Modul. Funktioniert ähnlich wie run.

### sessions
```bash
sessions
```
Zeigt alle aktiven Sessions an.

### sessions -i <ID>
```bash
sessions -i 1
```
Verbindet sich mit einer bestimmten Session.

### sessions -k <ID>
```bash
sessions -k 1
```
Beendet eine aktive Session.

### background
```bash
background
```
Legt die aktuelle Session in den Hintergrund.

### back
```bash
back
```
Verlässt das aktuell geladene Modul und kehrt zur Hauptkonsole zurück.

### save
```bash
save
```
Speichert die aktuellen Einstellungen dauerhaft.
