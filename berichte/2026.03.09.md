## Übung (VNC - Recherche)
### Angabe
Recherchiere Basics zu VNC (Virtual Network Computing).

### Recherche
**Kurzfassung:**
VNC ist eine Technologie, mit der man einen Computer über ein Netzwerk aus der Ferne steuern kann. Dabei sieht der Benutzer auf seinem eigenen Gerät den Desktop eines anderen Computers und kann diesen mit Maus und Tastatur bedienen, als würde er direkt davor sitzen. VNC arbeitet nach einem Client-Server-Prinzip: Auf dem entfernten Computer läuft ein VNC-Server, der den Bildschirminhalt über das Netzwerk sendet, während auf dem eigenen Gerät ein VNC-Client läuft, der den Bildschirm anzeigt und Eingaben überträgt. Die Kommunikation erfolgt über das Remote Framebuffer Protocol, bei dem hauptsächlich Bildschirmänderungen und Benutzereingaben übertragen werden. VNC funktioniert auf vielen Betriebssystemen wie Windows, Linux und macOS und kann sowohl im lokalen Netzwerk als auch über das Internet eingesetzt werden, es wird auch oft für IT-Support hergenommen, also Fernwartungen.


## Übung( VNC - Scan)

### Angabe
Scanne mit nmap den VNC Port auf Metasploitable. Teste Version und Banner. Verwende das nmap Script vnc-info.

### Lösung
Erstmal muss man schauen ob der Standart Port für VNC offen ist, das ist der Port 5900. 
Das macht man mit folgendem Befehl:

```
nmap -p 5900 192.168.96.131
```
192.168.96.131 das ist bei mir die IP Adresse von Metasploitable
