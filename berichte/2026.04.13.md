## Übung (netcat chat)
### Angabe
Verwende netcat um einen bidirektionalen Chat zwischen 2 unterschiedlichen Rechnern aufzubauen.

### Lösung
Ich habe auf meinem Labor PC die Ip Adresse 192.168.15.51 und war somit im Netz 15. Und Jonas war im Netz 16. Wir haben das mittels Kali Vmware gelöst und beide jeweils auf "bridged" gestellt. Ich habe auf meiner VM folgendes eingegeben: 
```
nc -lvp 4444
```
Mit diesem Befehl wird der Netcad Chat gestartet. 
Jonas hat auf seinem PC anfangs meine Ip Adresse gepinged um erstmal heraus zu finden ob er meinen PC im Netz finden kann. Dann hat Jonas auf seinem PC folgenden Befehl eingegeben: 
```
nc 192.168.15.51 4444
```
Anschließend besteht die Verbindung und beide können etwas in den Chat etwas hinein schreiben.

## Übung (netcat data send)
### Angabe
Übertrage, per netcat, den Inhalt einer lokalen Text-Datei an einen anderen Schüler (auf einem anderen Rechner). Soll am Zielsystem in einer Textdatei gespeichert werden.

### Lösung
Ich habe Jonas eine Text datei gesendet (datei.txt) in der folgender Text stand: Hallo Jonas du gehst in die HTL Braunau.     
Damit ich diese Datei senden konnte habe ich bei mir in der VM folgendes eingegeben:
```
nc 192.168.16.52 4444 < datei.txt
```

<img width="330" height="120" alt="image" src="https://github.com/user-attachments/assets/f512bbfb-6c53-42ea-95f4-9de00bdd0f3a" />
     
Die angegebene IP Adresse ist die IP Adresse von Jonas seiner VM.

Jonas hat um die Datei zu empfangen folgenden Befehl eingegeben:
```
nc -lvp 4444 > datei.txt
```
